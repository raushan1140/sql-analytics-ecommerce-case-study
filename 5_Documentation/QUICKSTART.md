# ⚡ Quick Start Guide

Get up and running in **5 minutes** with this step-by-step quick start!

---

## 🎯 Goal

View the 15 analytics queries and business insights from the Brazilian E-Commerce dataset.

---

## 📊 Fastest Path (5 Minutes)

### Step 1: Read Business Insights
1. Open file: **`4_Business_Insights/Business_Insights.md`**
2. Scroll through all 15 queries with findings
3. Done! ✅

**No setup required. No account needed.**

---

## 🔥 Interactive Path (30 Minutes)

Want to run queries yourself? Follow this:

### Step 1: Create Supabase Account (2 minutes)
```
1. Go to: https://supabase.com
2. Click "Start Your Project"
3. Sign up with GitHub or email
4. Verify email
```

### Step 2: Create Project (3 minutes)
```
1. Dashboard → New Project
2. Name: "ecommerce-analytics"
3. Create strong password
4. Click Create → Wait for init
```

### Step 3: Create Tables (5 minutes)
```
1. Go to SQL Editor
2. Create New Query
3. Copy code from: 3_ERD_and_Schema/Schema.sql
4. Paste & Run
5. Tables created! ✅
```

### Step 4: Import Data (15 minutes)
```
1. Go to Table Editor
2. Click "+" → Import Data
3. Choose each CSV from 1_Dataset/
4. Import all 9 files
5. Done! ✅
```

### Step 5: Run Queries (5 minutes)
```
1. Go to SQL Editor
2. New Query
3. Copy from: 2_SQL_Queries/Query_01_Revenue_by_Month.sql
4. Paste & Run
5. View results!
```

**Total time: 30 minutes**
